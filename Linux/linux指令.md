 
### list - 列出資料夾下內容

結構：
ls [OPTION] DIR

參數：
- `-l`：列出詳細檔案資訊 (權限、日期等)
- `-a`：也列出隱藏檔案
- `-R`：遞迴，也列出子資料夾內的所有檔案及資料夾內容
- `-S`: 按照檔案大小排序 
- `-t`: 按照修改時間排序 (由近至遠)

```
# 只列出檔名  
ls  
  
# 也列出隱藏檔案，外加詳細資訊  
ls -al  
  
# ls -al 的縮寫  
ll

# 可以搭配多個參數來顯示所需要的資訊
ls -laSrt
```

### change directory - 切換工作目錄

```
# 進入 myDir 資料夾  
cd myDir  
  
# 回上層目錄  
cd ..  
  
# 進入"家目錄"(相當於回到windows桌面的概念)  
cd ~  
  
# 進入"根目錄"  
cd /
```
